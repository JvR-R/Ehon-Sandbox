To add theme support to any page:

  1. Include theme.css FIRST in your HTML head
  2. Include other CSS files after theme.css
  3. Include theme-toggle.js before closing body tag
  4. Add the theme toggle button (or use border.php)
  5. Use CSS variables instead of hardcoded colors
